# Circuit Builder

An interactive digital circuit design and simulation tool built with React Flow for designing and testing logic circuits.

## What It Is

Circuit Builder is a web-based circuit design application that lets you visually create digital circuits using a node-based interface. You can add circuit components, connect them together, save your designs locally, and simulate the circuit behavior. The app provides a real-time canvas for circuit design with a component palette and comprehensive keyboard shortcuts for efficient workflow.

## Features

- **Visual Circuit Design** - Drag-and-drop interface using React Flow for building circuits
- **Component Library** - Pre-built circuit components with customizable properties
- **Circuit Simulation** - Simulate and test circuit behavior in real-time
- **Local Persistence** - Automatically save designs to browser local storage
- **Keyboard Shortcuts** - Full keyboard support for common operations
- **Dark Theme UI** - Modern dark interface with Tailwind CSS

## Tech Stack

- React 19 with TypeScript
- Vite for development and building
- React Flow for node-based diagram editing
- Zustand for state management
- Tailwind CSS for styling

## Getting Started

```bash
npm install
npm run dev
```

The app will be available at `http://localhost:5173`

## Build
